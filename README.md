# 功能说明
TradingWatcher是一款金融行情APP，提供全球股票、股指、外汇、期货、债券、数字货币等超十万种金融资产品种的实时行情报价，涵盖全球70多个国家和地区.

TradingWatcher支持多周期同屏K线回放,可模拟股票价格实时跳动,给您最真实的交易体验.

TradingWatcher更有订单流足迹图表,精准显示各价位成交量分布,帮助您快速精准判断市场行情.
